# Secrets App

Secrets website where you can upload your secrets anonymously. made in Dr.Angela Yu's Web development bootcamp.

## Built With

* [Node.js](https://nodejs.org/en/)
* [MongoDB](https://www.mongodb.com/)
* [EJS](https://ejs.co/)



## Installation
* npm
    ```bash
    npm install
    ```
